🔧 ステップ 1：Fedoraで修正
cd ~/app001_angry-type
# コードを編集
nano index.html
# または VS Code で
code .


修正が終わったら、保存してコミットする。
git add .
git commit -m "fix: ソース修正版アップロード"
☁️ ステップ 2：GitHubに上書きアップロード（push）
git push origin main
✅ これでGitHub上の app001-angry-type リポジトリが最新になる。
もしGitHubの認証が再度聞かれたら、**個人アクセストークン（PAT）**を使う。
トークンは https://github.com/settings/tokens
で作成。
（スコープは「repo」だけでOK）
